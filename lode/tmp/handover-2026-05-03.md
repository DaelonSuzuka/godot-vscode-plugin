## Breakthrough

Proved that the Python Lezer grammar pattern **works for GDScript** by creating `test-indent.grammar` — a minimal grammar that:
- Generates correctly with `lezer-generator`
- Produces all external token IDs: `indent`, `dedent`, `newline`, `blankLineStart`, `newlineBracketed`, `eof`
- Bundles successfully via Rollup into `dist/test-index.js` / `.cjs`

The key was following the Python grammar's structure exactly, especially:
1. `@context trackIndent from "./tokens.js"` declaration
2. `blankLine` as a **rule** in `@skip {}` (NOT just an external token)
3. `@skip {} { blankLine { blankLineStart space? Comment? (newline | eof) } }` pattern
4. `kw<term>` template for all keywords (avoids token overlap warnings)
